O que é SVG?

SVG significa Scalable Vector Graphics (Gráficos Vetoriais Escaláveis). Diferente de formatos de imagem como PNG ou JPG, que são baseados em pixels (raster), o SVG é baseado em XML e descreve imagens através de formas, caminhos e texto. Sua principal vantagem é a capacidade de ser redimensionado para qualquer tamanho sem perder qualidade, o que o torna perfeito para web design responsivo, ícones e, claro, animações.

Estilizando com CSS

Você pode estilizar SVGs usando atributos de apresentação (ex: `fill="blue"`) ou com CSS, o que é muito mais poderoso e flexível. As propriedades `stroke-dasharray` e `stroke-dashoffset` são a chave para criar as famosas animações de "desenho".

Demonstração Interativa do Efeito de Desenho

Como funciona:
1. `stroke-dasharray` cria um padrão de traços. Se o valor for o comprimento do círculo, teremos um único traço e um único espaço.
2. `stroke-dashoffset` desloca o início desse padrão. Animá-lo de um valor alto para 0 cria a ilusão de que a linha está sendo desenhada.

Criando Animações de Loading

Agora vamos juntar tudo! Abaixo estão alguns exemplos de animações de loading criadas com SVG e CSS. Analise o código de cada um para entender como as técnicas de `transform` e `stroke` são combinadas para criar efeitos fluidos e profissionais.

Spinner Simples

<!-- HTML --> <svg class="spinner" viewBox="0 0 50 50"> <circle class="path" cx="25" cy="25" r="20" fill="none" stroke-width="5"></circle> </svg> <!-- CSS --> <style> .spinner { animation: rotate 2s linear infinite; width: 50px; height: 50px; } .path { stroke: #4A90E2; stroke-linecap: round; animation: dash 1.5s ease-in-out infinite; } @keyframes rotate { 100% { transform: rotate(360deg); } } @keyframes dash { 0% { stroke-dasharray: 1, 150; stroke-dashoffset: 0; } 50% { stroke-dasharray: 90, 150; stroke-dashoffset: -35; } 100% { stroke-dasharray: 90, 150; stroke-dashoffset: -124; } } </style>

Checkmark de Sucesso

<!-- HTML --> <svg class="checkmark" viewBox="0 0 52 52"> <circle class="checkmark__circle" cx="26" cy="26" r="25" fill="none"/> <path class="checkmark__check" fill="none" d="M14.1 27.2l7.1 7.2 16.7-16.8"/> </svg> <!-- CSS --> <style> .checkmark__circle { stroke-dasharray: 166; stroke-dashoffset: 166; stroke-width: 2; stroke: #4A90E2; fill: none; animation: stroke 0.6s ease-out forwards; } .checkmark__check { stroke-dasharray: 48; stroke-dashoffset: 48; stroke-width: 2; stroke: #4A90E2; fill: none; animation: stroke 0.3s ease-out 0.6s forwards; } @keyframes stroke { 100% { stroke-dashoffset: 0; } } </style>

Pontos Pulsantes

<!-- HTML --> <svg height="24" width="80"> <circle class="dot" cx="15" cy="12" r="6" fill="#4A90E2"/> <circle class="dot" cx="40" cy="12" r="6" fill="#4A90E2"/> <circle class="dot" cx="65" cy="12" r="6" fill="#4A90E2"/> </svg> <!-- CSS --> <style> .dot { animation: pulse 1.5s ease-in-out infinite; } .dot:nth-child(2) { animation-delay: 0.2s; } .dot:nth-child(3) { animation-delay: 0.4s; } @keyframes pulse { 0%, 100% { opacity: 0; transform: scale(0.5); transform-origin: center; } 50% { opacity: 1; transform: scale(1.2); transform-origin: center; } } </style>
